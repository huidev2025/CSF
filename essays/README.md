# CSF 专栏书架 / The CSF Column

**简体中文** &#124; [English](README_en.md)

这里是 CSF（协作规范框架）的深度思想沉淀区。不仅记录了我们如何将 AI 真正作为“协作者”来对待的工程原理，也包含了整个方法论随实践不断自举而产生的哲学演进。

## 旗舰连载：《与智能共事》 / Flagship Series: Working with Intelligence

> “既然大模型的参数和窗口已经大到可以装下你整个代码库，为什么人机协作依然会偏离？因为你正用旧世界的物理规则，折磨着一个概率原生的新同事。”

本连载为 CSF 体系的 **v4.0 旗舰纲领**。它不再只是抽象的学术演绎，而是从四大系统（语义管理、协作通信、质量保障、演化系统）的严肃软件工程角度切入，通过高度具象的工程隐喻与极其扎实的理论架构，教你如何与这个“坏记性、概率原生、极其脆弱又极具活性”的智能生命共事。

## 几句关于阅读这组文章的说明 / Reading Notes

这套专栏书籍是 CSF 方法的**思想灵魂**——它不代表你上手使用所必须知道的标准操作（那是 [QUICKSTART.md](../QUICKSTART.md) 做的事情）。

如果你在阅读过程中被高密度的概念轰然劝退，完全不用焦虑：
* 先去复刻 [csf-minimal/](../csf-minimal) 进行 30 秒的 A/B 体验对比。
* 用这套规范去跟 AI 开发一个真实的小产品（例如哪怕几个文件的小程序）。
* 当你被 AI 迷失的上下文反复折磨、开始无奈骂它“智障”的时候，再回来读这个专栏中对应的篇章——你会发现每一条条文都是你“深夜血压飙升”后的血淋淋解药。
