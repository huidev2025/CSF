## 几句关于阅读这组文章的说明

这六篇是 CSF 这套方法长出来时的**思想原稿**——不是给读者重新组织的科普版，也不是给企业看的方案介绍。它的语言密度比仓库里其他文档高一些，需要慢读。

读不下去也没关系：先用 [QUICKSTART.md](../QUICKSTART.md)，等真的开始用 CSF 干活之后再回来读这些——很多句子要在你"自己撞过墙"之后才会显出意思。
