# dlog — 论文写作原始会话记录

> Raw dialogue logs from the paper writing process. Uncleaned. Contains Copilot tool traces and AI intermediate reasoning output.

---

## 说明

这些文件是论文写作过程中每次会话的**原始记录**。它们保留了：

- Owner 与 AI 之间的完整对话
- AI 工具调用痕迹（`Read`、`Searched`、`Created` 等行）
- 部分 AI 中间推理输出
- Obsidian 插件注入的 prompt 指令

这些噪音是原始记录的一部分，**刻意保留以供独立核查**。

如果你想找论文的修改过程，看 `_dlog_paper1_issues.md`（问题追踪日志），它按 issue 编号组织，有交叉引用。
