一、CSF Lite 是什么

CSF Lite 是 CSF 体系的中档——适合个人或小团队，在真实项目中落地人机协作。

你只需要： - 说出你的目标 - 与 AI 保持对话 - 听取建议、做决策

AI（参谋长）会帮你：理解需求 → 设计方案 → 规划任务 → 跟踪进度。

Clarity 是配套的桌面工具（可选），辅助编辑 CSF 文件和管理提示词。

三、准备工作

你需要三样东西：

3.1 获取 CSF Lite 文件包

从 GitHub releases 页面下载最新 zip 包，解压到你的项目文件夹。

不需要安装 Git，不需要克隆仓库——直接下载 zip 解压即可。

3.2 获取大模型 API

CSF Lite 支持所有主流大模型，你可以根据自己的预算和偏好选择。

推荐模型 特点
DeepSeek V4 Pro ⭐ 推荐 使用效果最佳，价格极其便宜
Claude Sonnet 4.6 使用效果佳，价格适中

以下以 DeepSeek 为例说明获取 API Key 的步骤。其他模型的操作类似，访问对应平台即可。

操作步骤（DeepSeek）：

  1. 打开浏览器，访问 DeepSeek 开放平台：https://platform.deepseek.com
  2. 注册账号（手机号或邮箱）
  3. 登录后进入「API Keys」页面
  4. 点击「创建 API Key」，命名（如 csf-lite），复制保存
  5. 进入「充值」页面，按用量充值（建议首次充 10~20 元试用）

⚠️ API Key 只显示一次，请立即保存到安全的地方。

费用参考：以上两个模型日常使用每月几元到几十元不等。

3.3 下载安装 VS Code（推荐）

VS Code 是微软出品的免费代码编辑器。CSF Lite 推荐使用 VS Code，但不是必须——任何支持 Copilot Chat 的编辑器都可以。

操作步骤：

  1. 打开浏览器，访问 VS Code 官网：https://code.visualstudio.com
  2. 点击「Download for Windows」（Mac 用户选择对应版本）
  3. 运行下载的安装程序，一路「下一步」完成安装
  4. 启动 VS Code

四、连接大模型

VS Code 的 Copilot Chat 是内置的，但默认使用 GitHub 的模型。要让 Copilot 使用你自己的 API，需要安装一个模型连接插件。

4.1 安装插件

  1. 打开 VS Code
  2. 点击左侧竖栏的「扩展」图标（或按 Ctrl+Shift+X）
  3. 搜索 DeepSeek V4 for Copilot Chat（如用 Claude 等其他模型，搜索对应插件）
  4. 点击「安装」

4.2 配置 API Key

安装完成后，在插件设置中填入你在 3.2 节获取的 API Key。

💡 一句话：Copilot Chat 已内置，只需装一个模型插件 + 填 Key，就可以用你自己的模型了。

五、开始使用 CSF Lite

5.1 打开项目

  1. 启动 VS Code
  2. 点击「File → Open Folder」，选择你解压的 CSF Lite 项目文件夹
  3. 点击左侧竖栏的「Copilot Chat」图标，打开 AI 对话窗口

5.2 开局

在 Copilot 聊天窗口中输入：

阅读 cos-context.md，开局

AI 会加载 CSF Lite 引擎，向你说一句「L1 brief」——介绍自己是谁、项目在什么状态、接下来建议做什么。

5.3 说出你的目的

开局后，直接告诉 AI 你想做什么。例如：

  • 「我想开发一个个人博客网站」
  • 「我想开发一个微信小程序 / 桌面应用 / Web 应用 / 手机 App」
  • 「我要做一个复杂系统的需求分析和架构设计」
  • 「帮我做一个项目报价书」
  • 「帮我写一个招标书 / 投标书」
  • 「我要整理一份市场调研报告」
  • 「帮我规划一个学习路线」

AI（参谋长）会理解你的目标，与你讨论方案、拆解任务、逐步推进。

5.4 保持沟通

  • 你说目的，AI 出方案 —— 你不需要自己拆解任务
  • AI 会主动提问 —— 不清楚的地方会问你
  • 你只需要做决策 —— 同意就继续，不同意就纠偏
  • 每次结束时说「88」或「收尾」 —— AI 会自动保存进度，下次接上

💡 核心要领：把 AI 当作平等的合作者，而不是问答机器。说清楚目的比说清楚步骤更重要。
